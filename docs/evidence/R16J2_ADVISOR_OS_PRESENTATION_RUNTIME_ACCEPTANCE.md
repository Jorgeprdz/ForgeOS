Date: 2026-07-16

## Outcome

The prospect-specific Quote-to-Sales Presentation execution authority is
`advisor-os/presentation/`. Manager OS retains only compatibility re-exports;
it has no presentation write authority. The browser bridge mounts the Advisor
OS context, prompt, slide-plan, review, editor, approval and export modules.

`R16I` is a superseded request identifier and is not the canonical tracking
identifier for this work.
