# Forge Mobile Widget Edit / Reorder / Collapse Scope 057O

Status: NEXT
Phase: 057O_MOBILE_WIDGET_EDIT_REORDER_COLLAPSE_SCOPE

## Goal

Define how Forge mobile widgets become configurable without losing the
mobile-first, fewer-clicks philosophy.

## Scope Candidates

- Long press widget edit mode.
- Reorder widgets.
- Collapse/expand widgets.
- Pin one widget above the fold.
- Hide non-essential widgets.
- Preserve Alfred, command bar, bottom nav, and context top nav.

## Non Goals

- No real CRM/calendar/message/file operation.
- No desktop changes.
- No final module implementation.
