## Before Designing A New Desktop Screen

- [ ] Identify module.
- [ ] Pick one template family.
- [ ] Define command intents.
- [ ] Define main workspace object.
- [ ] Define whether Alfred decision strip is needed.
- [ ] Define KPI / graph need.
- [ ] Define right rail role.
- [ ] Confirm mobile is not touched.

## Template Selection

- [ ] Command Table for dense row comparison.
- [ ] Document Workflow for quotes, policies and upload review.
- [ ] Analytics for charts, reports and goals.
- [ ] Record Detail for one selected client/policy/quote/opportunity.
- [ ] Review And Approval before any future action.

## Command Workspace

- [ ] Visible above fold.
- [ ] Contains primary intent.
- [ ] Has grouped suggestions if useful.
- [ ] Shows preview state.
- [ ] Uses safe action language.
- [ ] Does not execute.

## Alfred Decision Strip

- [ ] Explains what matters.
- [ ] Names who/what is affected.
- [ ] Explains why now.
- [ ] Provides one next preview action.
- [ ] Does not duplicate the table.

## Main Workspace

- [ ] Table columns have ownership.
- [ ] Labels do not split awkwardly.
- [ ] Row actions are readable.
- [ ] Empty/loading/error states are planned.
- [ ] Dense information remains scannable.

## Right Rail

- [ ] Only appears when useful.
- [ ] Never overlays main workspace.
- [ ] Collapses or moves below at compact widths.
- [ ] Contains context, limits, recommendation or feedback.
- [ ] Does not duplicate the hero or decision strip.

## Graphs

- [ ] Graph supports a decision.
- [ ] Labels are compact.
- [ ] Scale/context is understandable.
- [ ] Graph does not dominate first viewport unless analytics screen.

## Identity

- [ ] Alfred uses canonical bow tie.
- [ ] Profile avatar is separate.
- [ ] No letter mark for Alfred.
- [ ] No infinity mark for Alfred.
- [ ] Halo is subtle and purposeful.

## Safety

- [ ] Preview-only boundary is visible when actions appear.
- [ ] No send.
- [ ] No CRM write.
- [ ] No calendar create.
- [ ] No runtime execution.
- [ ] Human approval remains explicit.

## QA Viewports

- [ ] 1366x768.
- [ ] 1440x1000.
- [ ] 1536x864.
- [ ] 1920x1080.
